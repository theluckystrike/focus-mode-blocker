# Focus Mode - Blocker

A Chrome extension that blocks distracting websites to help users stay focused during work sessions.

## Status

**Phase 01** - Competitive Intelligence

## Tech Stack

- Chrome Extension Manifest V3
- HTML / CSS / JavaScript

## Project Structure

```
src/
├── popup/        # Extension popup UI
├── background/   # Service worker / background scripts
├── content/      # Content scripts injected into pages
└── assets/
    └── icons/    # Extension icons
docs/
└── research/     # Competitive intelligence & documentation
```
